Do you want report from Code metrics, but not using Visual Stusio 2010 Premium or Team Foundation Server 2010?

There is a simple solution:

Install Code Metrics Power Tool free of charge:

http://www.microsoft.com/download/en/details.aspx?id=9422

Install absolutely free extension from NuGet:

http://codemetricsviewer.wordpress.com/
